Хх
--

.. glossary::

    ХРОМ
        Кибервэр, особо очевидные аугментации.

    ХАКЕР
        Незаконно взаимодействующий с Матрицей, либо с помощью кибердеки, либо
        силой своего разума.

    ХЭЙ
        Фамильярная форма приветствия.

    ХОП
        Распространенное ругательство со ссылкой на человеческий зад.

    ХОШ
        Сорвать, испортить.
