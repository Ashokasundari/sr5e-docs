Бб
--

.. glossary::

    БАЗ
        Уходить.

    БОЧЁНОЧНИК
        Человек с большим количеством кибервэра, отсылка к части процесса
        трансплантации, во время которого пациент должен быть погружён в питательную
        смесь.

    БЕЗГРЕШНЫЙ
        Не имеющий ГРЕха человек. Для общества не существующий и полностью
        бесправный.
